## 1️⃣ 虚影(Ghost)的创建与位置计算

### 创建虚影 (BoardRow.ts 第 107-132 行)

```typescript
private setupDrag(card: CardView, item: SlotItem) {
  card.on('pointerdown', (e: FederatedPointerEvent) => {
    this.clearSlotTargetGlow();
    this.dragItem = item;
    this.dragging = card;
    
    // ⭐ 坐标转换：全局 → 局部
    const local = this.toLocal(e.global);
    this.dragOffset.x = local.x - card.x;
    this.dragOffset.y = local.y - card.y;

    // ⭐ 虚影创建
    this.dragGhost = new CardView(item);
    this.dragGhost.alpha = 0.7;                    // 半透明
    this.dragGhost.x = card.x;                     // 初始位置
    this.dragGhost.y = card.y;
    this.cardsLayer.addChild(this.dragGhost);      // Z轴：加到最上层

    // 原卡牌变透明
    card.alpha = 0.3;
    card.cursor = 'grabbing';

    // 绑定全局事件（重要：绑定到 stage，不是本容器）
    const stage = this.getStage();
    if (stage) {
      stage.eventMode = 'static';
      stage.on('pointermove', this.handleDragMove);
      stage.on('pointerup', this.handleDragEnd);
      stage.on('pointerupoutside', this.handleDragEnd);
    }
  });
}
```

**关键点**:
- `this.toLocal(e.global)` 将全局坐标转换为相对于 BoardRow 的局部坐标
- `dragOffset` 保存鼠标相对于卡牌的偏移
- 虚影 alpha=0.7 实现半透明效果
- 必须绑定到 stage，否则鼠标移出容器后无法跟踪

### 虚影跟踪鼠标 (BoardRow.ts 第 135-149 行)

```typescript
private handleDragMove = (e: FederatedPointerEvent) => {
  if (!this.dragGhost || !this.dragItem) return;
  
  // ⭐ 虚影跟随鼠标的核心公式
  const local = this.toLocal(e.global);
  this.dragGhost.x = local.x - this.dragOffset.x;
  this.dragGhost.y = local.y - this.dragOffset.y;

  // 计算悬停槽位并显示高亮
  if (this.isWithinOwnRow(e.global.y)) {
    const slotIdx = this.getSlotAtLocal(local.x);
    this.showHighlight(slotIdx, this.dragItem.size as 1 | 2 | 3);
  } else {
    this.clearHighlight();
  }

  // 回调给场景（用于显示卖出遮罩或储物箱高亮）
  this.onDragging?.(this.dragItem, e.global.x, e.global.y);
};
```

**公式解析**:
```
虚影X坐标 = toLocal(鼠标全局X).x - dragOffset.x
         = 本地鼠标X - (按下时的相对偏移)
         = 保持鼠标与卡牌的相对位置

例如：
- 鼠标全局 X = 100px
- BoardRow 全局 X = 20px，所以本地 X = 100 - 20 = 80px
- 原卡牌 X = 50px
- dragOffset.x = 80 - 50 = 30px
- 虚影 X = 80 - 30 = 50px (与原卡相同)
```

## 2️⃣ 槽位位置计算系统

### 槽位到屏幕坐标转换

```typescript
// BoardRow.ts 第 222-227 行
private getSlotAtLocal(localX: number): number {
  const step = CARD_UNIT + CARD_GAP;  // 88 + 6 = 94px
  const idx = Math.round(localX / step);
  if (idx < 0 || idx >= this._slotCount) return -1;
  return idx;
}
```

**示例计算**:
```
CARD_UNIT = 88px
CARD_GAP = 6px
step = 94px

槽位 0: X = 0 ~ 94px
槽位 1: X = 94 ~ 188px
槽位 2: X = 188 ~ 282px
...

localX = 100px → idx = Math.round(100/94) = 1 ✅
localX = 140px → idx = Math.round(140/94) = 1 ✅
localX = 180px → idx = Math.round(180/94) = 2 ✅
```

### 行Y坐标范围判断 (BoardRow.ts 第 198-201 行)

```typescript
private isWithinOwnRow(globalY: number): boolean {
  const rowGlobal = this.getGlobalPosition();
  return globalY >= rowGlobal.y - 15 && globalY <= rowGlobal.y + CARD_H + 15;
  //      ↑ 行顶部 - 容差           ↑ 行底部 + 容差
}
```

**容差说明**:
- `- 15` 和 `+ 15` 是容差范围
- 允许鼠标在行外 15px 时仍识别为在行内
- 改善用户体验（不需要完全对准）

### 显示高亮 (BoardRow.ts 第 236-246 行)

```typescript
private showHighlight(slotIdx: number, size: 1 | 2 | 3 = 1) {
  this.clearHighlight();
  if (slotIdx < 0) return;
  
  const w = cardWidth(size);  // 计算卡牌宽度
  this.highlightGraphic = new Graphics();
  this.highlightGraphic.roundRect(0, 0, w, CARD_H, 6);
  this.highlightGraphic.fill({ color: 0x4a90d9, alpha: 0.3 });
  this.highlightGraphic.stroke({ color: 0x4a90d9, width: 2 });
  this.highlightGraphic.x = slotIdx * (CARD_UNIT + CARD_GAP);
  this.addChild(this.highlightGraphic);
}

// 卡牌宽度计算 (layout.ts 第 8-10 行)
export function cardWidth(size: 1 | 2 | 3): number {
  return CARD_UNIT * size + CARD_GAP * (size - 1);
  // 1格: 88px
  // 2格: 88*2 + 6 = 182px
  // 3格: 88*3 + 6*2 = 276px
}
```

## 4️⃣ 拖拽释放与放置逻辑

### 完整的 handleDragEnd (BoardRow.ts 第 151-196 行)

```typescript
private handleDragEnd = (e: FederatedPointerEvent) => {
  // 1. 解绑事件
  const stage = this.getStage();
  if (stage) {
    stage.off('pointermove', this.handleDragMove);
    stage.off('pointerup', this.handleDragEnd);
    stage.off('pointerupoutside', this.handleDragEnd);
  }

  // 2. 恢复原卡牌
  if (this.dragging) {
    this.dragging.alpha = 1;
    this.dragging.cursor = 'grab';
  }

  // 3. 销毁虚影
  if (this.dragGhost) {
    this.cardsLayer.removeChild(this.dragGhost);
    this.dragGhost.destroy();
    this.dragGhost = null;
  }

  // 4. 清理高亮
  this.clearHighlight();
  this.clearSlotTargetGlow();

  // 5. 处理放置逻辑
  const item = this.dragItem;
  this.dragItem = null;
  this.dragging = null;

  if (item) {
    const gy = e.global.y;
    
    // 检查是否在本行内
    if (this.isWithinOwnRow(gy)) {
      const localInBoard = this.toLocal(e.global);
      const targetSlot = this.getSlotAtLocal(localInBoard.x);
      const targetItem = targetSlot >= 0 ? this.findItemCoveringSlot(targetSlot) : null;
      
      // 判断合成
      const canMerge = targetItem
        && targetItem.slotIndex !== item.slotIndex
        && targetItem.itemId === item.itemId
        && targetItem.tier === item.tier;
      
      if (canMerge) {
        this.onMerge?.(item, targetItem);
      } else if (targetSlot >= 0 && targetSlot !== item.slotIndex) {
        this.onSwap?.(item, targetSlot);
      }
    } else {
      // 拖出行外
      this.onDragOut?.(item, e.global.x, gy);
    }
  }

  this.onDragStop?.();
};
```

private handleBoardDragOut(item: SlotItem, gx: number, gy: number) {
  // 判断是否在储物箱范围
  if (this.stashOpen && gy >= Z2_Y && gy < Z2_Y + Z2_H) {
    const slot = this.stashRow.getSlotIndexAtGlobal(gx);

## 6️⃣ 层级管理（Z-Index）

### BoardRow 的容器结构 (BoardRow.ts 第 35-47 行)

```typescript
constructor(slotCount = 10) {
  super();
  this._slotCount = slotCount;

  // Z0: 背景槽位
  this.bgSlots = new Container();
  this.addChild(this.bgSlots);
  this.drawBgSlots();

  // Z1: 槽位发光层（目标高亮）
  this.slotGlowLayer = new Container();
  this.addChild(this.slotGlowLayer);

  // Z2: 卡牌层（原卡 + 虚影）
  this.cardsLayer = new Container();
  this.addChild(this.cardsLayer);  // ← 最后添加，显示在最上
}
```

**Z轴顺序**:
```
Z2 - cardsLayer
├─ 原卡牌 (alpha=0.3)
└─ 虚影卡牌 (alpha=0.7)  ← 后添加，显示在上

Z1 - slotGlowLayer
└─ 高亮框

Z0 - bgSlots
└─ 背景槽位
```

**为什么虚影在上？**
- `addChild` 按顺序添加
- 最后添加的元素在最上面
- 虚影比原卡晚添加 → 显示在原卡之上

## 9️⃣ CardView 的创建与渲染

### CardView 构造函数 (CardView.ts 第 9-23 行)

```typescript
export class CardView extends Container {
  private itemId: string;
  private tier: string;
  private size: ItemSize;

  constructor(item: SlotItem) {
    super();
    this.itemId = item.itemId;
    this.tier = item.tier;
    this.size = item.size;
    this.draw();
    
    const w = cardWidth(this.size);
    this.eventMode = 'static';
    this.hitArea = new Rectangle(0, 0, w, CARD_H);  // 触发区域
  }

  private draw() {
    const cfg = gameState.itemsMap.get(this.itemId);
    const w = cardWidth(this.size);
    const h = CARD_H;

    // 1. 边框
    const border = new Graphics();
    border.roundRect(-2, -2, w + 4, h + 4, 8);
    border.fill({ color: TIER_COLORS[this.tier] ?? 0x555555, alpha: 0.9 });
    this.addChild(border);

    // 2. 主体
    const body = new Graphics();
    body.roundRect(0, 0, w, h, 6);
    body.fill(TIER_BG[this.tier] ?? 0x222233);
    this.addChild(body);

    // 3. 名称
    const name = new Text({
      text: cfg.name,
      style: { 
        fill: '#ffffff', 
        fontSize: this.size === 1 ? 11 : 13, 
        fontFamily: 'Arial', 
        fontWeight: 'bold' 
      },
    });
    name.anchor.set(0.5, 0);
    name.x = w / 2;
    name.y = 4;
    this.addChild(name);
    
    // ... 其他信息
  }
}
```
