## 💻 代码集成示例

### 基础 AudioManager 框架
```typescript
// client/src/audio/AudioManager.ts
import { Howl } from 'howler';

export class AudioManager {
  private sounds: Map<string, Howl> = new Map();
  private enabled = true;
  private volume = 0.7;

  async init() {
    // 预加载音效
    this.preloadSound('battle:card-trigger', '/assets/sounds/battle/card-trigger.mp3');
    this.preloadSound('battle:damage', '/assets/sounds/battle/damage.mp3');
    // ... 更多音效
  }

  private preloadSound(key: string, src: string) {
    const howl = new Howl({ src });
    this.sounds.set(key, howl);
  }

  playEffect(key: string) {
    if (!this.enabled) return;
    const sound = this.sounds.get(key);
    if (sound) {
      sound.volume(this.volume);
      sound.play();
    }
  }

  setVolume(vol: number) {
    this.volume = Math.clamp(vol, 0, 1);
  }

  toggle() {
    this.enabled = !this.enabled;
  }
}
```

## ⚠️ 常见问题

### Q: 该用 howler.js 还是 Web Audio API？
**A**: 推荐 howler.js
- ✅ 跨浏览器兼容性好
- ✅ 音量/淡入淡出控制
- ✅ 音效复用共享
- ✅ TypeScript 支持

### Q: 音效文件在哪里放？
**A**: `public/assets/sounds/` (Vite 会自动处理)

### Q: 如何避免音效过多导致卡顿？
**A**: 
- 预加载关键音效 (in `AudioManager.init()`)
- 使用 MP3 (较小体积)
- 限制同时播放数

### Q: 音效怎么禁用？
**A**: 添加设置项
```typescript
audioManager.toggle();  // 开/关
audioManager.setVolume(0.5);  // 调节音量
```
