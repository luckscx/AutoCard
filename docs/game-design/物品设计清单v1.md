# AutoCard 物品设计清单 v1

> 整理时间：2026/04/19
> 设计思路：以大巴扎 991 条原始数据为灵感源，结合 AutoCard 的 4 英雄体系（战士/法师/盗贼/牧师）重新设计。
> 目标：60 个设计完善的物品，每个英雄 3 条清晰 build 路线。

---

## 设计原则

1. **每个物品必须有明确的使用场景**，知道它在哪条 build 里用
2. **bronze tier 的物品要足够有吸引力**，玩家愿意升级而非直接卖掉
3. **端口 value 按 tier 分级**：bronze / silver / gold / diamond，每级约 1.5x 成长
4. **通用物品不超过 20 个**，其余按英雄分配，保持英雄差异化

---

## 端口类型说明

| 端口 | 说明 | 常用 build |
|---|---|---|
| damage | 直接伤害 | 战士核心、法师辅助 |
| poison | 持续剧毒 | 盗贼核心 |
| burn | 持续灼烧 | 法师核心 |
| shield | 护盾 | 牧师核心 |
| heal | 治疗 | 牧师辅助 |
| haste | 加速自身或相邻 | 所有 build 的加速辅助 |
| charge | 充能（触发特效用） | 法师/工具型 |
| slow | 减速敌方 | 盗贼辅助 |
| freeze | 冻结敌方 | 战士爆发辅助 |
