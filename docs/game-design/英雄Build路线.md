# 英雄 Build 路线设计

> 为6个英雄各设计 3 条核心 build 路线，确保每局体验差异化。
> 含推荐卡组（6-8张）、核心联动逻辑、通用卡搭配建议和平衡性验证。

## 一、斯泰尔（stelle）— 机甲/战士型

**基础属性**：HP 100 (+10/级) | 起始金币 10 | 起始卡牌：蒸汽汤勺

### Build A：冻结爆发

**核心思路**：冻结敌人 → 冻结期间高伤爆发 → 永冻控制链

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | W01 冰爪 | damage(30)+freeze(1) | 冻结+伤害一体，冻结态×1.5 |
| 爆发 | W02 霜刃巨斧 | damage(80)+freeze(2) | 冻结后伤害翻倍，终结技 |
| 增强 | W04 永冻护腕 | 被动 | 冻结持续时间+1s，容错率大增 |
| 增强 | W05 极寒圣物 | 被动 | 每次冻结→下次攻击+40伤害 |
| 通用 | U12 冰封地雷 | freeze(2) | 补充冻结来源 |
| 通用 | U07 沙漏 | haste(2) | 加速冰爪和巨斧更快触发 |
| 通用 | U14 木盾 | shield(20) | 冻结CD间隙的生存保障 |

**联动逻辑**：冰爪/冰封地雷提供冻结 → 永冻护腕延长冻结 → 极寒圣物叠加伤害 → 霜刃巨斧终结
**弱点**：依赖冻结触发，遇到高抗性/清debuff的敌人会乏力；前期缺生存

### Build B：叠层强化

**核心思路**：每场战斗永久叠加伤害/护盾 → 越打越强 → 拓荒者之心翻倍上限

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | W06 血战长剑 | damage(25) | 击杀永久+5伤害(上限+50) |
| 核心 | W08 符文双刃 | damage(40)+crit | 暴击永久+10暴伤(上限+100) |
| 增强 | W07 战场纪念章 | 被动 | 每场胜利全武器+3伤害 |
| 增强 | W09 成长护甲 | 被动 | 每场+10护盾叠加不清零 |
| 上限 | W10 拓荒者之心 | 被动 | 叠层上限翻倍 |
| 通用 | U05 旧皮带 | 被动 | 相邻物品CD-1 |
| 通用 | U18 生命护符 | 被动 | HP上限+20 |

**联动逻辑**：血战长剑+战场纪念章双线叠伤 → 成长护甲叠加护盾 → 拓荒者之心翻倍上限 → 符文双刃暴击乘区
**弱点**：前期极弱，需要2-3天发育；怕速攻/被提前击杀

### Build C：重击暴击

**核心思路**：大体积武器+暴击 → 一击必杀型 → 暴击回血续航

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | W11 破天巨锤 | damage(120)+crit25% | 慢速重击核心输出 |
| 增强 | W12 战神铠甲 | 被动 | 每个size3物品+10%暴击率 |
| 增强 | W13 暴怒护符 | 被动 | HP<50%全武器+30%暴击率 |
| 续航 | W14 鲜血宝石 | 被动 | 暴击回10HP |
| 终结 | W15 屠龙之刃 | damage(100) | 对HP>自身目标+30%伤害 |
| 通用 | U07 沙漏 | haste(2) | 加速巨锤CD |
| 通用 | U19 战场急救箱 | heal(40)+shield(20) | 生存保障 |

**联动逻辑**：破天巨锤慢速高伤 → 战神铠甲/暴怒护符堆暴击率 → 鲜血宝石暴击回血 → 屠龙之刃收割
**弱点**：巨锤CD长(14s)，空窗期危险；依赖暴击率，脸黑时输出不稳定

## 二、杜利（dooley）— 法师/机器人型

**基础属性**：HP 100 (+10/级) | 起始金币 10 | 起始卡牌：皂沫中士

### Build A：灼烧叠层

**核心思路**：叠灼烧层数 → 层数够高后引爆 → 持续灼烧+爆发双伤害

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | M01 火焰法杖 | burn(8) | 稳定灼烧来源 |
| 爆发 | M03 烈焰宝珠 | burn(20) | 灼烧≥3层时爆炸+60伤害 |
| 增强 | M04 炎魔之戒 | 被动 | 灼烧每秒+2 |
| 收割 | M05 凤凰羽毛 | 被动 | 灼烧结束时HP<30%→30爆发伤害 |
| 通用 | U07 沙漏 | haste(2) | 加速灼烧触发频率 |
| 通用 | U11 减速陷阱 | slow(3) | 减速敌方延缓进攻 |
| 通用 | U14 木盾 | shield(20) | 灼烧持续期间的生存 |

**联动逻辑**：火焰法杖持续灼烧 → 炎魔之戒强化 → 烈焰宝珠引爆 → 凤凰羽毛收割残血
**弱点**：灼烧怕清除/高回复对手；前期伤害不足

### Build B：充能爆发

**核心思路**：高频充能 → 满3层释放大招 → 充能溢出缩短全队CD

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | M06 奥术晶球 | charge(1) | 满3层→100伤害 |
| 充能 | M07 能量棱镜 | charge(1) | 高频充能(CD4s) |
| 辅助 | M09 共鸣水晶 | 被动 | 相邻触发→充能+1 |
| 增强 | M08 超载法阵 | 被动 | 充能满→burn+slow+damage |
| 加速 | M10 魔力溢出符 | 被动 | 充能清零→全物品CD-2s |
| 通用 | U09 齿轮发条 | charge(1) | 额外充能来源 |
| 通用 | U14 木盾 | shield(20) | 生存 |

**联动逻辑**：能量棱镜+齿轮发条高频充能 → 共鸣水晶邻位联动 → 奥术晶球爆发 → 魔力溢出符加速循环
**弱点**：充能期无输出，被速攻打死；需要精密摆位

### Build C：连锁反应

**核心思路**：一次触发引发连锁 → 复制效果 → 滚雪球式爆发

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | M11 连锁闪电杖 | damage(30) | 40%概率跳跃+15伤害 |
| 核心 | M12 镜像魔典 | damage(50) | 复制相邻物品效果 |
| 辅助 | M13 召唤图腾 | damage(40) | 放置时召唤复制品 |
| 增强 | M14 时空涟漪 | 被动 | 每次触发全物品value+1 |
| 加速 | M15 奥术超载符文 | 被动 | 充能上限-1(更快爆发) |
| 通用 | U07 沙漏 | haste(2) | 加速连锁触发 |
| 通用 | U14 木盾 | shield(20) | 生存 |

**联动逻辑**：连锁闪电杖触发 → 时空涟漪全物品+1 → 镜像魔典复制效果 → 滚雪球
**弱点**：需要多物品联动，单卡质量低；40%跳跃概率不稳定

## 三、瓦内莎（vanessa）— 盗贼/武器型

**基础属性**：HP 95 (+9/级) | 起始金币 10 | 起始卡牌：毒液

### Build A：剧毒叠层

**核心思路**：叠加毒 → 毒伤×减速协同 → 被动毒溢出伤害

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | R01 蛇牙匕首 | poison(5) | 极速施毒(CD4s) |
| 核心 | R02 毒雾飞镖 | poison(8)+slow(1) | 毒+减速，减速时毒×1.3 |
| 增强 | R03 毒液蒸馏器 | poison(20) | 毒持续时间+2s |
| 增强 | R04 暗影毒瓶 | 被动 | 每层毒→下次攻击+5 |
| 开局 | R05 炼毒师腰带 | 被动 | 战斗开始自动毒5/s×5s |
| 通用 | U11 减速陷阱 | slow(3) | 减速配合毒伤加成 |
| 通用 | U07 沙漏 | haste(2) | 加速施毒频率 |

**联动逻辑**：炼毒师腰带开局毒 → 蛇牙匕首/毒雾飞镖叠毒 → 减速陷阱增强毒伤 → 暗影毒瓶伤害溢出
**弱点**：毒伤怕清除；对高回复对手效果打折

### Build B：灼烧控制

**核心思路**：减速+灼烧双控 → 减速增强灼烧 → 灼烧叠伤

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | R06 烙刀 | damage(20)+burn(5) | 减速时伤害和灼烧×1.5 |
| 辅助 | R07 皮皮虾 | damage(20)+burn(5) | 减速时灼烧+2s |
| 控制 | R08 烟雾弹 | slow(4) | 减速期间+15%暴击率 |
| 叠伤 | R09 火枪 | damage(60) | 灼烧触发时伤害永久+10 |
| 增强 | R10 点火手套 | 被动 | 30%概率附加burn(3/s)×3s |
| 通用 | U12 冰封地雷 | freeze(2) | 补充控制链 |
| 通用 | U14 木盾 | shield(20) | 生存 |

**联动逻辑**：烟雾弹减速 → 烙刀/皮皮虾灼烧增强 → 火枪叠伤 → 减速期间暴击加成
**弱点**：依赖减速触发，缺减速时输出打折；控制链断裂时脆弱

### Build C：暴击连击

**核心思路**：高频攻击+暴击 → 暴击叠加暴击 → 低HP极限输出

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | R11 双持飞刀 | damage(15)+crit20% | 极速攻击核心 |
| 核心 | R12 影刃 | damage(20)+crit15% | 连续暴击+30%暴伤 |
| 增强 | R13 暗杀徽章 | 被动 | 暴击→下次CD-0.5s |
| 增强 | R14 幸运兔脚 | 被动 | 全局暴击率+15% |
| 终结 | R15 刺客长袍 | 被动 | HP每低10%→+5%暴击率 |
| 通用 | U07 沙漏 | haste(2) | 加速飞刀 |
| 通用 | U16 急救包 | heal(15) | 低HP续航 |

**联动逻辑**：双持飞刀+影刃高频攻击 → 幸运兔脚/暗杀徽章堆暴击 → 暴击缩CD→更多暴击 → 刺客长袍低血爆发
**弱点**：身板脆(HP95)，低血冒险打法容错低；暴击率不稳定

## 四、皮格马利翁（pygmalien）— 牧师/坦克型

**基础属性**：HP 120 (+12/级) | 起始金币 12 | 起始卡牌：没药

### Build A：护盾墙

**核心思路**：超厚护盾 → 护盾反伤 → 护盾存在时全队增益

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | P01 圣光盾 | shield(40) | 护盾碎→20反伤 |
| 核心 | P02 神圣壁垒 | shield(80) | 护盾存时全CD-15% |
| 增强 | P03 荆棘圣衣 | 被动 | 受伤时回30%护盾值 |
| 增强 | P04 信仰护符 | 被动 | 护盾上限+30，碎盾heal(10) |
| 增强 | P05 圣甲虫 | 被动 | 护盾>100→全物品+20%效果 |
| 通用 | U14 木盾 | shield(20) | 补充护盾 |
| 通用 | U13 破甲箭 | damage(15) | 穿透伤害(护盾墙缺输出) |

**联动逻辑**：圣光盾+神圣壁垒→超厚护盾→圣甲虫触发全队增益→荆棘圣衣+信仰护符碎盾回血
**弱点**：伤害严重不足，纯被动输出；被穿透/破甲克制

### Build B：治疗续航

**核心思路**：持续治疗 → 治疗溢出转护盾 → 血量上限堆叠

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | P06 治愈之光 | heal(25) | 溢出治疗→护盾 |
| 核心 | P07 圣水法阵 | heal(40) | 每次治疗+5(上限+50) |
| 爆发 | P08 生命之泉 | heal(60) | HP越低治疗×2 |
| 再生 | P09 再生戒指 | 被动 | 每秒回3HP |
| 辅助 | P10 天使翎羽 | 被动 | 治疗→自身50%回血 |
| 通用 | U16 急救包 | heal(15) | 补充治疗 |
| 通用 | U18 生命护符 | 被动 | HP上限+20 |

**联动逻辑**：再生戒指持续回血→治愈之光溢出转护盾→圣水法阵治疗叠层→生命之泉低血爆发回
**弱点**：同样缺输出；纯续航打不死人，需要偷输出手段

### Build C：护盾加速

**核心思路**：护盾+加速双端口 → 前排盾辅助后排快速输出 → 护盾越厚加速越强

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | P11 守护之光 | shield(30)+haste(1) | 盾+加速一体 |
| 核心 | P12 圣洁圣所 | shield(40)+haste(2) | 有盾时全队+20%触发速度 |
| 辅助 | P13 时光弦月 | haste(3) | 护盾越厚CD越短 |
| 持续 | P14 牧师法袍 | 被动 | 每2s→最右盾(15) |
| 加速 | P15 神圣协调 | 被动 | 护盾触发→下次CD-1s |
| 通用 | U07 沙漏 | haste(2) | 补充加速 |
| 通用 | U13 破甲箭 | damage(15) | 输出补充 |

**联动逻辑**：守护之光→盾+加速→圣洁圣所全队增益→时光弦月/神圣协调加速循环→牧师法袍持续盾
**弱点**：输出最低的build之一，完全依赖队友物品输出；需要摆位配合

## 五、朱尔斯（jules）— 欢乐/另类型

**基础属性**：HP 90 (+9/级) | 起始金币 10 | 起始卡牌：巴努叶

### Build A：欢乐加速

**核心思路**：全队加速 → 加速触发护盾 → 被加速物品效果+25%

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | J01 欢乐铃铛 | haste(2)→all | 加速全队，首触发翻倍 |
| 联动 | J02 快乐滤镜 | 被动 | 加速时→护盾8 |
| 爆发 | J03 派对礼花 | damage(15)+haste(3) | 伤害+加速双端口 |
| 循环 | J04 疯狂节拍器 | haste(1) | 邻位触发→自身CD-1s |
| 增强 | J05 欢乐颂歌 | 被动 | 被加速物品+25%效果 |
| 通用 | U07 沙漏 | haste(2) | 补充加速 |
| 通用 | U14 木盾 | shield(20) | 生存 |

**联动逻辑**：欢乐铃铛全队加速→快乐滤镜每次加速得盾→欢乐颂歌+25%效果→节拍器自循环加速

### Build B：美食护盾

**核心思路**：护盾+治疗双防 → 护盾期间减伤20% → 溢出治疗转护盾

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | J06 奶油泡芙 | heal(15)+shield(15) | 治疗+护盾双端口 |
| 核心 | J07 果酱面包 | shield(20) | 护盾时每秒回3HP |
| 爆发 | J08 巧克力堡垒 | shield(60) | 碎盾时heal(20) |
| 开局 | J09 糖衣护甲 | 被动 | 每场开始盾15 |
| 减伤 | J10 甜蜜庇护所 | 被动 | 有盾时伤害-20% |
| 通用 | U14 木盾 | shield(20) | 补盾 |
| 通用 | U16 急救包 | heal(15) | 补治疗 |

**联动逻辑**：糖衣护甲开局盾→甜蜜庇护所-20%伤害→果酱面包盾期间回血→巧克力堡垒碎盾回血

### Build C：暴击盛宴

**核心思路**：高频攻击+灼烧暴击 → 暴击叠暴击 → 朱尔斯专属物品越多暴击率越高

| 位置 | 卡牌 | 端口 | 作用 |
|------|------|------|------|
| 核心 | J11 胡椒粉碎机 | damage(12)+crit20% | 高速攻击+暴击灼烧 |
| 核心 | J12 辛辣炸弹 | burn(15)+crit15% | 暴击时灼烧×2 |
| 增强 | J13 厨师徽章 | 被动 | 第3次暴击→全队+10%暴击 |
| 增强 | J14 完美调味 | 被动 | 暴击伤害+50%/暴击率-10% |
| 终结 | J15 终极盛宴 | 被动 | 每张朱尔斯卡+8%暴击率 |
| 通用 | U07 沙漏 | haste(2) | 加速粉碎机 |

### 设计观察

1. **所有英雄的前后期体验有差异**：叠层/成长型前期弱后期强（stelle叠层、mak炼金），爆发型前后期均衡（stelle冻结、vanessa暴击），防御型前期稳后期缺输出（pygmalien全部、jules美食护盾）

2. **英雄间无绝对最优**：每个英雄有3条差异化路线，路线间有真实取舍（稳定vs爆发、前期vs后期、简单vs复杂）

3. **跨界组合空间大**：通用物品(20个)是跨英雄build的桥梁，如减速陷阱同时服务vanessa毒流和mak冰霜试剂

4. **需关注的平衡点**：
   - pygmalien 所有build输出不足，可能需要通用输出卡补充
   - jules HP最低(90)但无逃命手段，美食护盾build需重点测试生存
   - mak 与 vanessa 的毒流、mak 与 dooley 的灼烧流有重叠，需要通过细节差异化（mak侧重药剂叠层/冰霜组合，vanessa侧重减速联动/暴击，dooley侧重引爆/连锁）
