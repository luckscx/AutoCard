## 8. 坐标计算速查表

### 棋盘格子 X 坐标
```
slotIndex = 0: x = 24
slotIndex = 1: x = 24 + 94 = 118
slotIndex = 2: x = 24 + 188 = 212
...
slotIndex = 9: x = 24 + 846 = 870

通用公式: x = INNER_X + slotIndex * (CARD_UNIT + CARD_GAP)
              = 24 + slotIndex * 94
```

### 卡牌宽度
```
size = 1: width = 88
size = 2: width = 182 (= 88*2 + 6*1)
size = 3: width = 276 (= 88*3 + 6*2)

通用公式: width = CARD_UNIT * size + CARD_GAP * (size - 1)
                = 88 * size + 6 * (size - 1)
```

### 卡牌高度 (固定)
```
普通棋盘: CARD_H = 110
```

### 拖拽时的关键 Y 值
```
从棋盘拖起并向上 → 超过 Z3_Y (340) 时触发卖出
从储物箱拖起并向上 → 超过 Z2_Y (106) 时触发卖出

判断在某区域内:
isInZ1 = globalY >= 2 && globalY < 102
isInZ2 = globalY >= 106 && globalY < 336
isInZ3 = globalY >= 340 && globalY < 496
isInZ4 = globalY >= 500 && globalY < 600
```
